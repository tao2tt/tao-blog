## 💡 算法思想对比

| 算法思想 | 核心思想 | 适用场景 | 时间复杂度 |
|---------|---------|---------|-----------|
| **分治** | 分而治之，合并结果 | 归并排序、快速排序 | O(nlogn) |
| **贪心** | 局部最优 → 全局最优 | 活动选择、霍夫曼编码 | O(nlogn) |
| **动态规划** | 最优子结构 + 重叠子问题 | 背包问题、最长公共子序列 | O(n²) |
| **回溯** | 试错 + 剪枝 | N 皇后、数独、排列组合 | O(2ⁿ) |

## 📝 学习建议

### 1. 理解算法思想

- 不要死记硬背代码
- 理解算法的核心思想和适用场景
- 掌握算法的时间/空间复杂度分析

### 2. 刷题实践

- 从简单题开始，逐步提升难度
- 每道题尝试多种解法
- 总结解题模板和套路

### 3. 对比学习

- 对比不同算法的优缺点
- 理解为什么在某些场景下选择特定算法
