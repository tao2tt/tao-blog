## 💡 学习建议

### 1. 学习方法

```
理解原理 → 手写实现 → 刷题巩固 → 总结归纳
```

### 2. 时间分配

- **理解原理**：30% 时间
- **手写代码**：30% 时间
- **刷题练习**：40% 时间

### 3. 刷题策略

```
第 1 遍：看答案理解
第 2 遍：独立实现
第 3 遍：优化代码
第 4 遍：举一反三
```

**网站：**

## 📝 学习计划

### 第 1 阶段：基础入门（2 周）

**目标：** 掌握基础数据结构

**Week 1：**
- 复杂度分析（大 O 表示法）
- 数组与链表（实现与应用）
- 栈与队列（实现与应用）

**Week 2：**
- 哈希表（原理与实现）
- 字符串处理
- 基础算法（排序、查找）

### 第 2 阶段：进阶提升（3 周）

**目标：** 掌握高级数据结构

**Week 3-4：**
- 树与二叉树（遍历、BST、AVL）
- 堆与优先队列
- 并查集

**Week 5：**
- 图论基础（DFS、BFS、最短路径）
- 跳表与 B 树（了解）

### 第 3 阶段：算法突破（4 周）

**目标：** 掌握核心算法思想

**Week 6：**
- 递归与分治
- 贪心算法

**Week 7-8：**
- 动态规划（重点）
- 回溯算法

**Week 9：**
- 滑动窗口
- 双指针
- 位运算

### 第 4 阶段：实战冲刺（3 周）

**目标：** 面试准备

**Week 10-11：**
- LeetCode 高频 100 题
- 按类型刷题（数组、链表、树、DP...）

**Week 12：**
- 模拟面试
- 总结归纳
- 查漏补缺
