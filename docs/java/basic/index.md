## 💡 学习建议

1. **动手实践**：每个知识点都要写代码验证，不要只看不练
2. **源码阅读**：集合框架、并发包要多看源码，理解设计思想
3. **对比学习**：比如 HashMap vs ConcurrentHashMap，synchronized vs ReentrantLock
4. **性能意识**：学习时要关注性能影响，比如集合选型、锁的粒度
