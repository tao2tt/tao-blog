## 📘 阶段一：Java 基础夯实（第 1-2 月）

**目标：** 打牢基础，能独立写高质量代码

| 周次 | 学习内容 | 执行方式 | 产出 |
|------|----------|----------|------|
| 1-2 | Java 核心语法、集合框架、异常处理 | 每日 1h 阅读《Effective Java》+ coding | 笔记 + 代码示例 |
| 3-4 | 多线程与并发（Thread、Lock、Concurrent 包） | 每日 1h 学习 + 写 Demo 验证 | 并发工具类 Demo |
| 5-6 | JVM 基础（内存模型、GC、类加载） | 每周 3h 学习 + 用 VisualVM 分析 | JVM 调优笔记 |
| 7-8 | 复习 + LeetCode 刷题（每日 2 题） | 刷题 + 总结 | 50 题 AC |

**推荐资源：**
- 📚 《Effective Java》
- 📚 《Java 并发编程实战》
- 📺 B 站「尚硅谷 Java 并发」

**推荐资源：**

## 💡 关键建议

1. **利用现有工作：** ES 同步、订单查询等工作就是最好的实践场景，边工作边学习

2. **AI 方向选择：** 建议走 **AI 应用开发** 路线（而非算法），Java 背景 + AI 应用能力是市场稀缺人才

3. **别贪多：** 按阶段来，每个阶段完成再进入下一个，不要同时学太多

4. **记录成长：** 本博客每周写点学习心得，既是总结也是履历

5. **主动沟通：** 每季度和 leader 聊一次职业发展，争取更多成长机会

6. **保持行业敏感度：** 关注 AI、云原生、Serverless 等趋势
