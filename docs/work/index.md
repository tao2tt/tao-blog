## 💡 写作建议

1. **技术方案**：重点写清楚为什么选这个方案，对比过哪些方案
2. **项目复盘**：诚实记录问题和教训，方便以后回顾
3. **性能优化**：记录优化前后的数据对比，有图有真相
4. **故障排查**：记录排查思路和工具使用，形成知识沉淀
