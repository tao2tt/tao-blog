## 💡 学习建议

### 1. 理论与实践结合
- 先了解大模型基本原理（Transformer、Attention）
- 然后快速进入应用层开发（API 调用、Prompt 工程）
- 不要一开始就深入数学推导，容易劝退

### 2. 从简单到复杂
- 先用现成 API 做简单应用（聊天机器人）
- 再学习 RAG 增强模型能力
- 最后学习 Agent 自主决策

### 3. 关注最新进展
- AI 领域发展极快，每周都有新工具
- 关注 Twitter、知乎上的 AI 博主
- 多逛 Hugging Face 看热门项目

### 4. 项目驱动学习
- 每学一个技术点就做个小 Demo
- 最终整合成完整项目
- 把项目部署上线，写博客记录
